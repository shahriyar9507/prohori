# Third-Party Attribution

PRAHARI is built on open data and open-source software. All third-party resources used are listed here, as required by the SciBlitz AI Challenge 2026 rulebook (§10.2). This file is updated as dependencies are added.

| Resource | Provider | License / terms |
|---|---|---|
| GDELT Event Database | The GDELT Project | Open, free to use |
| Sentinel-1 SAR imagery | ESA / Copernicus Data Space Ecosystem | Copernicus open licence |
| xView3-SAR dataset | DIU / xView | Research / open |
| AIS vessel positions | aisstream.io | Free tier, per site terms |
| Global Fishing Watch | Global Fishing Watch | CC BY-NC 4.0 / API terms |
| UN Comtrade | United Nations | Open data policy |
| Open-Meteo weather | Open-Meteo | CC BY 4.0 |
